🎮 Battle Craft 3D - Fighter Showdown
Fast-paced 3D fighting game built with Three.js featuring unique fighters, special attacks, mobile controls, and arcade-style combat.

🥊 About
Battle Craft 3D is a browser-based fighting game where players choose a fighter and battle AI opponents in an intense 1v1 arena. The game features stylized 3D characters, special abilities, health and energy systems, smooth animations, and mobile-friendly controls.
